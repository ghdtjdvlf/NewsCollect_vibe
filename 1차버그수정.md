1. 버그 수정 (Critical Bug Fixes)
[커뮤니티] DCinside 링크 오류: 게시글 클릭 시 상세 페이지가 아닌 메인 페이지로 이동하는 현상 수정.

[검색] 썸네일 누락: 검색 결과 리스트에서 뉴스 썸네일이 노출되지 않는 문제 해결.

[콘텐츠] 뉴스 본문 로딩 불안정:

특정 뉴스(게임뷰, YTN, 조선일보, 한겨레 등)에서 본문 및 이미지가 파싱되지 않는 문제 개선.

이미지 로딩 성공률 제고 (크롤링 로직 및 예외 처리 강화).

[성능] 탭 이동 및 데이터 로딩 정지: 탭 전환 시 혹은 검색 이후 데이터가 불러와지지 않는 현상 (데이터 오버플로우 또는 메모리 누수 확인 필요).

[카테고리] 정렬 기능: 카테고리 클릭 시 뉴스 필터링/정렬이 작동하지 않는 버그 수정.

[카테고리] 태그 동기화: 뉴스에 포함된 모든 태그를 카테고리 목록에 자동 추가.

[스크롤] 무한 스크롤 에러: 최신 뉴스 하단 스크롤 시 발생하는 런타임 에러 수정.

2. UI/UX 및 기능 개선 (Enhancements)
[애니메이션] 전환 효과 변경: 뉴스 클릭 시 '통통 튀는 모션' 제거 → ease-in 애니메이션으로 교체.

[상세보기] 아코디언/모달 제어: 새 뉴스 클릭 시 기존에 열려 있던 뉴스는 자동으로 닫히도록 설정.

[커뮤니티] 소스 확장: 기존 DCinside 외에 '펨코(fmkorea)' 추가 수집.

[AI 요약] 3줄 요약 기능 도입:

'상세보기' 버튼을 '3줄 요약'으로 변경.

클릭 시 AI가 원문 분석 후 요약 제공.

하단 문구: "AI가 뉴스 원문을 파악하고 요약합니다."

로딩 문구: "야무지게 요약 중이에요..."

[정보 표시] 데이터 정확도:

우측 조회수 아이콘(eye)에 실제 뉴스 조회수 연동.

'화제 뉴스'의 시간 표시를 '뉴스 작성일' 기준으로 변경.

각 뉴스 카드에 해당 언론사 로고(심볼) 표시.

[레이아웃] 그리드 뷰 추가: 리스트 형태(주제, 뉴스, 제목, 시간)와 카드 형태(사진 포함)를 선택할 수 있는 그리드/리스트 토글 기능.
