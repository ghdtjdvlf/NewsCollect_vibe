# Liquid News KR — PRD v1.1

> 한국 실시간 뉴스 모바일 애그리게이터

---

## 1. 제품 개요

네이버·다음·구글뉴스와 디시·에펨·클리앙에서 뉴스를 수집·정제해 모바일에 최적화된 형태로 제공하는 뉴스 애그리게이터.

| 항목 | 내용 |
|------|------|
| 플랫폼 | 모바일 웹 (iOS / Android) |
| 핵심 기능 | 최신뉴스 · 실시간 화제뉴스 · 검색 |
| 데이터 소스 | 네이버 · 다음 · 구글뉴스 + 디시 · 에펨코리아 · 클리앙 |
| 기술 스택 | Next.js · Framer Motion · GSAP · Firecrawl MCP · Playwright |

---

## 2. 3대 핵심 카테고리

### 🔥 실시간 화제뉴스

현재 한국에서 가장 뜨거운 이슈를 커뮤니티 반응 기반으로 선별.

**파이프라인**

```
뉴스 포털 수집 → 커뮤니티 교차 필터링 → 화제뉴스 피드 표시
```

| 단계 | 소스 | 수집 대상 |
|------|------|-----------|
| ① 수집 | 네이버 뉴스 | 섹션별 많이 본 뉴스, 댓글 많은 뉴스 |
| ① 수집 | 다음 뉴스 | 실시간 이슈 키워드 연동 뉴스 |
| ① 수집 | 구글 뉴스 (KR) | AI 토픽 클러스터링 결과, 중복 제거 참고 |
| ② 필터 | 디시인사이드 · 에펨 · 클리앙 | 인기 게시글 키워드 교차 대조 |

---

### 📰 최신뉴스

경제·사건사고 분야 최신 기사 실시간 수집.

| 소스 | 수집 카테고리 |
|------|--------------|
| 네이버 뉴스 | 경제 · 사건사고 · 사회 |
| 다음 뉴스 | 경제 · 사회 · 사건 |
| 구글 뉴스 (KR) | 경제 · 주요 뉴스 |

---

### 🔍 검색

키워드 입력 시 3개 포털 동시 검색 후 통합 표시.

- 동일 토픽 중복 기사 클러스터링 → 대표 기사 1개로 표시
- 최신순 / 관련도순 정렬
- 검색어 기반 관련 키워드 추천

## 3. 화면 구성 (모바일)

- **하단 탭바**: 🔥 화제 · 📰 최신 · 🔍 검색 3탭 고정
- **공통 헤더**: 스크롤 시 Liquid 블러 강화
- **뉴스 카드**: 썸네일 · 카테고리 뱃지 · 제목 · 출처 · 시간 / 클릭 시 Liquid 확장
- **검색 화면**: 상단 인풋 · 최근 검색어 · 통합 결과 리스트

## 4. 기술 스택

| 분류 | 기술 | 용도 |
|------|------|------|
| 프레임워크 | Next.js 14 (App Router) | 모바일 웹 + SSR |
| 애니메이션 | Framer Motion + GSAP | Liquid 전환 · Blob 배경 · 스크롤 효과 |

| 스타일링 | Tailwind CSS + Glassmorphism | Apple 글래스모피즘 디자인 |

## 5. SEO 전략

### 기술 SEO

| 항목 | 구현 방법 | 우선순위 |
|------|-----------|----------|
| SSR / ISR | 뉴스 상세 SSR, 카테고리 목록 ISR (재검증 60초) | P0 |
| 동적 메타데이터 | 뉴스 제목 기반 title · description 자동 생성 (`generateMetadata`) | P0 |
| OG / Twitter Card | 썸네일 + 제목으로 og:image 자동 생성 (Next.js ImageResponse) | P1 |
| Sitemap / robots.txt | 동적 sitemap.xml 자동 생성, 크롤러 허용/차단 경로 분리 | P1 |
| Core Web Vitals | LCP < 2.5s · CLS < 0.1 · INP < 200ms, 이미지 next/image 최적화 | P1 |
| JSON-LD | NewsArticle Schema → 구글 뉴스 상단 노출 및 리치 스니펫 | P2 |

### 콘텐츠 SEO

- **URL 구조**: `/news/[category]/[slug]` (예: `/news/경제/코스피-2600`)
- **H1 태그**: 뉴스 제목 그대로 사용, 중복 H1 금지
- **카테고리 페이지**: "오늘 경제 뉴스", "실시간 사건사고" 등 롱테일 키워드 타깃
- **canonical**: 동일 기사 다중 URL 중복 색인 방지
- **내부 링크**: 같은 토픽 기사끼리 연결 → 체류시간 및 크롤 깊이 향상

### 모바일 SEO

- `viewport` meta 필수: `width=device-width, initial-scale=1`
- PWA `manifest.json` → 홈 화면 추가 시 앱처럼 실행
- `next/font`로 FOUT 방지, 시스템 폰트 폴백 적용
- 터치 타겟 최소 44×44px 보장
